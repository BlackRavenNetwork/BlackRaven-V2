BlackRaven (BLKR) v2 — chain design (locked for transition)
============================================================

Base: NeoxaChain/Neoxa (MIT). Independent GitHub repo (not a GitHub "Fork" link).

Identity
--------
  Name:     BlackRaven
  Ticker:   BLKR
  Website:  https://blackraven.network/

Consensus / economics
---------------------
  Algorithm:              KawPoW
  Max supply:             32,000,000,000 BLKR
  Initial block reward:   5,000 BLKR (base subsidy before split)
  Block time:             ~1 minute
  Halving interval:       ~3,200,000 blocks (~6 years at 1 min)
  Dev fee:                0.5% of block subsidy (50 basis points), forever
                          — same % after every halving (fee = subsidy * 0.005)
                          — used for exchange listings, promotion, ecosystem
  Dev fee payout:         separate coinbase output (FounderPayment)
  Smartnodes:             DISABLED (fSmartnodesEnabled=false; no ProTx, no SN reward %)
  Miner share:            99.5% of block subsidy (remainder after 0.5% dev fee)
  InstantSend/ChainLocks: OFF (no LLMQ quorums without smartnodes)
